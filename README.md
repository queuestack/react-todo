# React Todo
React를 활용한 Todo List

## Features
* [✓] 새로운 TODO(제목 + 내용)를 작성한다
* [✓] 사용자의 선택에 의해 TODO에는 마감 기한을 넣을 수 있다.
* [✓] 우선순위를 조절할 수 있다.
* [✓] 완료 처리를 할 수 있다.
* [✘] 마감기한이 지난 TODO에 대해 알림을 노출한다.
* [✓] TODO 목록을 볼 수 있다.
* [✓] TODO 내용을 수정할 수 있다.
* [✓] TODO 항목을 삭제할 수 있다.

## Additional Features
* [✓] Unit Test : 
* [✘] Integration Test
